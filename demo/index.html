TOC, the Angular way

ng-toc generates a dynamic table of contents for online documentation, articles, blogs, and more.

It's a snap to add it to your project:

  1. Inject the ngToc module into your Angular app.
  2. Add the ng-toc attribute to your content area.
  3. Add ng-toc-list to the element that should contain the TOC.

Save your energy!

Focus on content, not manually building document navigation.

ng-toc collects heading elements specified in the ng-toc attribute — e.g.: ng-toc="h1, h2" — or defaults to H1, H2, and H3 elements if no attribute is given.

Go right there

The ng-toc-list directive provides an unordered list of direct links to each heading.

Style away

The ng-toc-list template provides a barebones presentation of the data. It's easy to style it any way you like.
